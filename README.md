# RosterBolt

[![CI Status](https://github.com/AY2526S2-CS2103T-T12-1/tp/workflows/Java%20CI/badge.svg)](https://github.com/AY2526S2-CS2103T-T12-1/tp/actions)

![Ui](docs/images/Ui.png)

RosterBolt is a desktop contact and roster management application for volunteer coordinators who need to plan manpower for recurring events.

It is optimized for users who prefer fast, keyboard-driven workflows over mouse-heavy interactions. RosterBolt helps reduce repetitive administrative work when handling medium to large volunteer pools.

## Why RosterBolt

- **Keyboard-first usage** for faster data entry and command execution.
- **Efficient bulk-oriented workflows** to cut down repetitive contact management tasks.
- **Offline-capable desktop app** for use in low-connectivity environments.
- **Local data handling** to better support privacy-conscious users.

For detailed documentation, see the [RosterBolt Product Website](https://ay2526s2-cs2103t-t12-1.github.io/tp/).
