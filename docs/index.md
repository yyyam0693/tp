# RosterBolt

[![CI Status](https://github.com/AY2526S2-CS2103T-T12-1/tp/workflows/Java%20CI/badge.svg)](https://github.com/AY2526S2-CS2103T-T12-1/tp/actions)

![Ui](images/Ui.png)

**RosterBolt is a desktop application for volunteer coordinators managing recurring event manpower and volunteer contacts.** While it has a GUI, most user interactions are optimized for a fast CLI (Command Line Interface) workflow.

RosterBolt is designed for coordinators who:

* Manage contact pools of 20 to 500 volunteers
* Work independently
* Prefer desktop, keyboard-first workflows over mouse-heavy interactions
* Are comfortable with CLI-style usage
* Need privacy-conscious, offline-capable operations
* Want to avoid slow, repetitive tasks such as editing contacts one-by-one

Key value provided by RosterBolt:

* Faster command-driven contact and roster management
* Reduced repetitive admin effort through efficient workflows
* Reliable usage in environments without Internet access
